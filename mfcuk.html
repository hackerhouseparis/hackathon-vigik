Utiliser mfcuk

Une puce NFC dispose de 12 blocs, protégés pour chacun par 2 clés.
Pour faire un dump d'une puce NFC dont les "clés" de sécurité des "blocs" ne sont pas celle pas défaut, on utilise mfcuk.

Pour installer mfcuk:

  • mkdir -p /tmp/compil
  • cd /tmp/compil
  • wget https://github.com/nfc-tools/mfcuk/archive/master.zip
  • unzip master.zip
  • cd mfcuk-master
  • autoreconf -vis
  • ./configure
  • make
  • sudo make install
  • /usr/local/bin/mfcuk
  • mfcuk

Et lancer mfcuk -C -R 0 -v 1 -s 250 -S 250 -o dump.bin
